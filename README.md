This is an MCP server for demo purposes.
Please specify the startup command on the MCP client side as follows:

```
Example:
command: "uv",  # or "/Users/your-username/.local/bin/uv"
args: [
        "--directory",
        "/ABSOLUTE/PATH/TO/PARENT/FOLDER/interview-mcp",
        "run",
        "screening.py",
      ]
```
